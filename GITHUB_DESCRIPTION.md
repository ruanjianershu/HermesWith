# HermesWith - GitHub 项目简介

## 简短描述（Short description）

多租户 AI 智能体管理平台，支持智能体生命周期管理、任务调度和企业级安全控制。

## 详细描述（Long description）

### 🏢 企业级智能体管理平台

HermesWith 是一个开源的多租户 AI 智能体管理平台，专为需要安全、可扩展智能体服务的企业设计。

### ✨ 核心能力

- **多租户隔离** - 公司级别的数据隔离，API Key 认证
- **智能体生命周期管理** - 创建、配置、监控、删除智能体
- **任务调度系统** - 优先级队列，异步执行，状态追踪
- **企业安全** - Fernet 加密，审计日志，速率限制

### 🏗️ 技术栈

- **后端**: Python 3.11+, FastAPI, SQLAlchemy 2.0
- **数据库**: PostgreSQL 15+, Redis 7+
- **安全**: JWT, API Key, Fernet 加密
- **部署**: Docker, Docker Compose

### 🚀 适用场景

- 企业 AI 自动化平台
- 多团队智能体服务共享
- 任务编排和执行系统
- AI 工作流管理

### 📊 项目统计

- 35+ Python 模块
- 完整的测试覆盖
- 详尽的 API 文档
- Docker 一键部署

## 英文版本

### Short Description

Multi-tenant AI agent management platform with agent lifecycle management, task scheduling, and enterprise-grade security controls.

### Long Description

**HermesWith** is an open-source multi-tenant AI agent management platform designed for enterprises requiring secure, scalable agent services.

**Key Features:**
- 🔐 Multi-tenant isolation with company-based data separation
- 🤖 Full agent lifecycle management (create, configure, monitor, delete)
- 📋 Priority task queue with async execution and status tracking
- 🔒 Enterprise security: Fernet encryption, audit logging, rate limiting

**Tech Stack:** Python 3.11+, FastAPI, SQLAlchemy 2.0, PostgreSQL, Redis, Docker

**Use Cases:** Enterprise AI automation, multi-team agent service sharing, task orchestration, AI workflow management
